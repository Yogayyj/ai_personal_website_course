第2课：用 AI 生成第一个网站

让学生快速做出第一个网站雏形，建立项目成就感。

本节课学生会完成

  • 使用 AI 生成个人网站初版
  • 创建网站基本页面结构
  • 替换个人姓名、简介和项目内容
  • 在电脑上预览网页效果
  • 初步理解网页由哪些部分组成

课堂内容

本节课不会要求学生从零手写代码，而是让学生通过 AI 生成一个可以运行的网站。老师会带学生学习如何把自己的需求描述清楚，并让 AI 根据需求生成网页。

GitHub 是什么？

GitHub 是一个基于 Git 的代码托管平台。你可以把网页代码、图片、文案等内容放在 GitHub 上进行保存、协作和版本管理。

  • 像“云端硬盘”一样存文件：网站文件不会只存在本地电脑里。
  • 像“时间机器”一样看历史：每次修改都可以记录，方便回退到之前版本。
  • 方便分享：你可以把项目链接发给老师、同学或招生官查看。

为什么我们用 GitHub 发布静态网站？

  • 免费：对课堂项目非常友好，适合学生快速起步。
  • 部署简单：使用 GitHub Pages 可以把 HTML/CSS/JS 网站直接发布到线上。
  • 稳定且规范：有助于学生形成标准化的项目管理习惯。
  • 便于长期维护：后续更新作品、文案时只需继续提交和同步。

详细流程：GitHub 账号注册（课堂可投屏带做）

  1. 打开官网：访问 https://github.com/，点击右上角 Sign up。
  2. 填写邮箱：建议使用长期使用、可稳定接收验证码的邮箱。
  3. 设置密码：至少 8 位，建议包含大小写字母与数字。
  4. 创建用户名（Username）：尽量专业、易识别，建议用英文名或姓名拼音（例如 alex-chen）。
  5. 完成人机验证：按页面提示完成验证流程。
  6. 邮箱验证：进入邮箱点击 GitHub 验证链接，完成账号激活。
  7. 选择个性化设置：学生可先选择免费方案（Free），其余选项保持默认即可。
  8. 登录后基础设置：补充头像、个人简介（Bio）和地区信息，便于后续展示个人形象。

什么是 Markdown 文件？

Markdown 是一种轻量级文本格式，常见文件后缀是 .md。它可以用非常简单的符号写出结构清晰的文档。

  • 常见用途：项目说明（README）、学习笔记、任务清单、更新日志。
  • 为什么课程中会用：你可以用 Markdown 写网站说明、项目介绍，方便老师和同学快速阅读。
  • 示例：# 标题、## 小节、- 列表项、[链接文字](链接地址)。

README.md 示例（项目说明模板）

下面是一个可以直接给学生使用的 README 模板。学生可以先复制，再根据自己的申请方向和项目经历做个性化修改。

# Personal Website Project

## Project Overview

This is my personal website for college application and portfolio presentation.

## Purpose

The website is designed to showcase my academic interests, personal projects, creative work, and future goals.

## Website Sections

- Home
- About Me
- Projects
- Resume
- Contact

## Tools Used

- HTML
- CSS
- GitHub
- AI Assistant

## What I Learned

Through this project, I learned how to plan a website, edit HTML and CSS, use GitHub to manage code, and publish my work online.

## Future Improvements

- Add more projects
- Improve mobile design
- Add videos and images
- Connect a custom domain

课堂总结：这几个工具分别是什么角色？

  • HTML 是网页本身。
  • CSS 是网页的外观。
  • Markdown 是项目的说明文档。
  • GitHub 是保存和展示这些内容的平台。

也可以用更生动的比喻帮助学生记忆：

  • HTML/CSS 是“作品”。
  • README.md 是“作品说明书”。
  • GitHub 是“作品展厅”。

AI 指令示例

请帮我生成一个适合高中生申请大学使用的个人网站。
风格简洁、现代、有科技感。

网站需要包含：
1. 首页 Hero Section
2. About Me
3. Projects
4. Resume
5. Contact

请使用清晰的排版和适合申请展示的内容结构。

当堂产出

  • 一个网站初版
  • 一个首页页面
  • 基础导航栏
  • 自我介绍区域
  • 项目展示区域
  • 联系方式区域

课后任务

  • 替换成自己的英文名字
  • 写出自己的申请方向
  • 添加至少一个项目标题
  • 准备项目图片或相关素材
