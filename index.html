课程介绍

Build first. Refine and Learn.

AI个人网站设计工作坊

在大学申请中，学生不仅需要成绩、文书和活动经历，也需要一个能够集中展示自己特点、兴趣方向和项目成果的个人平台。 相比零散的材料，一个设计完整、能够在线访问的个人网站，可以更直观地呈现学生的个人形象、学术兴趣、项目经历和成长轨迹。

个人网站是一种更现代、更专业的展示方式。它不仅能通过文字、图片、视频、音频等多媒体形式展示学生“做过什么”， 也能体现学生如何组织信息、表达想法、呈现项目，并将一个想法真正落地成可访问的作品。 对计划申请计算机、设计、工程、数据、商科或跨学科方向的学生来说，这种能力本身就是非常有价值的申请补充。

本课程将带领学生结合 AI 工具、基础前端知识与 GitHub 工作流，从零开始搭建一个真正可以上线的个人网站。 课程不会从复杂的 HTML、CSS、JavaScript 底层知识点开始，而是采用“边做边学”的方式。 学生将在老师引导下，像完成一个真实项目一样，不断提出需求、修改页面、优化视觉、整理项目素材、管理代码并最终上线自己的作品。

课程核心理念

在这门课里，他们会从真实项目入手，把 AI 变成自己的设计助手和开发伙伴， 逐步完成一个能够展示个人故事、项目经历和申请方向的个人网站

不从代码细节开始，而从真实作品开始。
Start with a real project, not endless code details.

课程目标

学生最终不仅能拥有一个网站，更重要的是学会如何表达自己、展示项目、熟练使用AI工具实现技术落地。

完成个人网站

学生将拥有一个属于自己的个人网站，包含首页、自我介绍、项目展示、简历信息和联系方式。

学会 AI 协作

学生将学习如何向 AI 提出清晰需求，并通过不断提问、修改和优化，完成网站设计与开发。

建立申请展示意识

学生将理解大学申请中如何展示兴趣、能力、项目经历和成长过程，而不仅仅是罗列活动。

整理个人项目

学生会学习如何把普通项目经历整理成 Portfolio 内容，包括背景、过程、成果和反思。

掌握上线流程

学生将注册并使用 GitHub，将自己的网站上传、管理并最终发布上线。

提升审美表达

学生将学习排版、配色、留白、图片使用和整体风格统一，让网站更像正式作品。

适合学生

本课程适合正在准备大学申请、想展示个人项目、作品或兴趣方向的学生，也适合没有编程基础但愿意动手实践的学生。

工具清单

课程会使用主流且适合学生上手的工具，帮助他们完成从内容整理、AI 生成、代码管理到网站上线的完整流程。

GitHub

用于创建账号、保存代码、管理个人网站项目，并展示技术学习过程。

AI 工具

用于生成网站代码、修改页面、优化文字、生成项目介绍和调整视觉风格。

Vercel / GitHub Pages

用于网站部署和上线，获得可以分享给他人的网站链接。

Figma

用于分析网页布局、设计页面草图、整理视觉风格。

Google Docs / Notion

用于写自我介绍、整理项目材料、保存网站文案和准备申请展示内容。

浏览器开发环境

用于预览网页效果，检查页面展示，并进行基础调试。
